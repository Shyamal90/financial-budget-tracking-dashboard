# Financial Budget Tracking Dashboard

## Objective
To design and develop a Financial Budget Tracker Dashboard using Excel and SQL that helps the finance team monitor departmental budgets, track actual spending, identify overspending areas, and make data-driven financial decisions through interactive reports and a visual dashboard.

## Problem Statement
The finance team was facing difficulty in:
- Tracking budget allocation across departments
- Monitoring actual expenses in real time
- Identifying overspending and budget variance
- Generating quick financial insights for decision-making
- Maintaining centralised and structured financial data

## Key Features
- Budget vs Actual Analysis
- Department-wise Expense Tracking
- Overspending Alerts
- Monthly & Quarterly Trend Analysis
- KPI Cards for Total Budget, Total Spend, Variance
- Interactive Filters and Visual Charts
- SQL Queries for data extraction and analysis

## Tools Used
- Excel: Dashboard creation, Pivot Tables, Charts, KPI Cards
- SQL: Data cleaning, aggregation, analysis, reporting queries

## KPIs
- Total Budget
- Total Expense

## Insights
- August and October exceeded the budget
